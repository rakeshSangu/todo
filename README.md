# 📝 Todo App

A simple and interactive Todo List application built using **ReactJS**. Users can add, delete, and mark tasks as completed with a clean and responsive UI.

---

## 🚀 Features

### ✅ Core Features:
- **Add Task**: Users can enter a task name and add it to the list.
- **Delete Task**: Remove tasks from the list with a delete button.
- **Mark as Completed**: Clicking a task marks it as completed with a strike-through effect.
- **Live Task Updates**: The task list updates in real time.
- **State Management**: Utilizes React’s `useState` hook to store tasks.

### 🎨 Design & User Experience:
✔ **Clean and Modern UI** – Intuitive and visually appealing design.  
✔ **Fully Responsive** – Works seamlessly across mobile, tablet, and desktop screens.  
✔ **Easy to Use** – Minimalist design with clear action buttons.  
✔ **User-Friendly Feedback** – Completed tasks have strikethrough effects and dimmed colors.  

---

## 🛠 Tech Stack
- **ReactJS** → For building the UI and handling state.
- **CSS** → For styling and responsiveness.
- **React Hooks** → `useState` for managing the task list.
